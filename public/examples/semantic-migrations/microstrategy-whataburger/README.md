# Whataburger-style MicroStrategy benchmark

This simulated bundle mirrors documented Strategy REST response families: project metadata, cube/report definitions, and a dashboard definition containing chapters, pages, filters, datasets, and visualizations. It is reconstructed from OmniKit's Whataburger-style demo and is not a production MicroStrategy export or brand-owned dataset.

Use it to test deterministic normalization before AI translation. A high score does not prove metric-result parity, prompt answers, selectors, security filters, derived elements, report limits, or visual fidelity; those remain explicit validation and review requirements.
