# Simulated Whataburger Power BI Project

This is a synthetic, source-control-friendly Power BI Project fixture. It is not a production export, customer data, or brand-owned content.

The fixture follows the documented PBIP topology closely enough to exercise:

- a `.pbip` project descriptor;
- folder-based TMDL semantic definitions;
- M/Power Query expression evidence;
- relationship keys and RLS evidence;
- a split PBIR report with report, page, and visual JSON files; and
- optional nested Workspace Scanner metadata.

It exists to validate project assembly and migration evidence. It does not certify DAX results, Power Query execution, security parity, or visual parity.
